# simuing-lab
simuing site
<br/>

### 개발환경
- Meteor Framework
  - MongoDB
  - React
  - typescript
- add package
  - fourseven:scss
  - underscore
<br/>


#### 구현기능  
>방명록 등록/삭제 기능 구현  
<br/>

#### 진행중
>방명록 수정 기능 구현  
>헤더 디자인
<br/>

#### 프로젝트 설치 과정
> create project
``` javascript 
meteor create --typescript simuing-lab
```
<br/>

> add scss
``` javascript 
meteor add fourseven:scss
```
<br/>

> add underscore
``` javascript 
meteor add underscore
```
